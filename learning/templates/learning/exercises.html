Esercizio 1: GET Request

Problema da risolvere: Ottieni la lista di tutti gli utenti dall'API.

import requests

response = requests.get('/api/users')
data = response.json()
print(data)
Suggerimento: Usa il metodo get di requests.

Esercizio 2: POST Request

Problema da risolvere: Crea un nuovo post.

import requests

data = {'name': 'Mario', 'email': 'mario@example.com'}
response = requests.post('/api/users', json=data)
print(response.json())
Suggerimento: Usa il metodo post e passa i dati come JSON.

Esercizio 3: PUT Request

Problema da risolvere: Aggiorna un post esistente.

import requests

data = {'name': 'Mario Rossi'}
response = requests.put('/api/users/1', json=data)
print(response.json())
Suggerimento: Usa il metodo put per aggiornare.

Esercizio 4: DELETE Request

Problema da risolvere: Elimina un post.

import requests

response = requests.delete('/api/users/1')
print("Status Code:", response.status_code)
Suggerimento: Usa il metodo delete.
