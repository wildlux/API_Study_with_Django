üåê Come Funziona un'API

Le API (Application Programming Interfaces) sono il ponte che permette a diverse applicazioni di comunicare tra loro. Immagina di essere in un ristorante: il cliente (tu) ordina dal menu, il cameriere (API) porta l'ordine in cucina (server), e torna con il cibo (dati).

In termini tecnici, un'API definisce un insieme di regole e protocolli che permettono a un software di interagire con un altro. Le Web API utilizzano il protocollo HTTP per trasferire dati, tipicamente in formato JSON.

Esplora le sezioni interattive qui sotto per approfondire:

üì§ Esempio Pratico:

Tu chiedi: "Dammi la lista degli utenti"
Server risponde: [Mario, Laura, Giuseppe]

1 Il Client fa una RICHIESTA
"Voglio i dati degli utenti"
2 Il Server ELABORA
Cerca i dati nel database
3 Il Server RISPONDE
Invia i dati in formato JSON

üîß Esempi di Richieste API

GET /api/users

Ottieni tutti gli utenti

import requests

response = requests.get('/api/users')
data = response.json()
print(data)

POST /api/users

Crea un nuovo utente

import requests

data = {'name': 'Mario', 'email': 'mario@example.com'}
response = requests.post('/api/users', json=data)
print(response.json())

PUT /api/users/1

Aggiorna utente esistente

import requests

data = {'name': 'Mario Rossi'}
response = requests.put('/api/users/1', json=data)
print(response.json())

DELETE /api/users/1

Elimina utente

import requests

response = requests.delete('/api/users/1')
print('Utente eliminato')
