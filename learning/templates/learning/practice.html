üìö Esercizio Pratico: GET Request
+

Obiettivo: Fare una richiesta GET a un'API pubblica e visualizzare i dati.

import requests

# Definizione dell'URL
url = "https://jsonplaceholder.typicode.com/users"

# Esecuzione della richiesta GET
response = requests.get(url)

# Verifica dello status code
if response.status_code == 200:
    # Stampa della risposta
    print(response.json())
else:
    # Stampa dell'errore
    print("Errore nella richiesta:", response.status_code)

üéØ Esercizio 4: Sicurezza nelle API
+

Le API spesso richiedono autenticazione. Ecco come gestire richieste con API Key.

import requests

# Esempio con autenticazione
headers = {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
}

response = requests.get('https://api.example.com/data', headers=headers)
print(response.json())

üîê Concetti di sicurezza:

  • API Keys: Chiavi di accesso univoche
  • OAuth: Protocollo di autorizzazione
  • HTTPS: Comunicazione criptata
  • Rate Limiting: Limite richieste per prevenire abusi
